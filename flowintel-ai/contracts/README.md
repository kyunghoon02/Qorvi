# Agent Contracts

This directory defines the structured inputs and outputs for Qorvi AI
agents.

Initial contracts:

- `wallet-analyst`
- `signal-explainer`
- `alert-briefing`
- `interactive-analyst`
- `background-detection-agent`
- `finding-object-schema`

Initial spec:

- `wallet-analyst.md`
- `signal-explainer.md`
- `alert-briefing.md`
- `interactive-analyst.md`
- `background-detection-agent.md`
- `finding-object-schema.md`

Every contract should prefer:

- Evidence-backed inputs
- Explicit coverage window metadata
- Clear separation between observed facts and inferred interpretations
- Structured JSON output over free-form prose
